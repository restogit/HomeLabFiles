Switches/Routers config, use different IP addresses:
enable
conf t
hostname <devicename>
interface <fast or gig> <interface number>
ip address <ip address> <subnet mask>
exit
enable password <password>
username <username> secret <password>
line vty 0 15
login local
transport input all
exit
ip domain-name <domain name>
crypto key generate rsa
2048
ip ssh version 2
end
write

Enable telnet debug:
debug telnet

Switch interface configuration is different.
By configuring VLAN 1 with an IP address,
there is no need to individually assign IPs to the wired interfaces:

conf t
int vlan 1
ip address <ip address> <subneet mask>
no shut
end
